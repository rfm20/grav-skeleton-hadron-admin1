## Karten
Dieses Modul ist ein Beispiel für die Verwendung von Karten. Es kann verwendet werden, um verschiedene Informationen oder Funktionen in einem kompakten Format darzustellen. Jede Karte kann ein Bild, einen Titel, einen Text und einen Link enthalten, der zu einer anderen Seite führt.
