## SbS Liste (abwechselnde Seiten)
Die SbS Liste ist eine Sammlung von SbS Elementen, die abwechselnd auf der linken und rechten Seite positioniert sind. Jedes Element kann ein Bild, einen Titel, Text und einen Link enthalten. Die Textausrichtung kann in 9 verschiedenen Positionen erfolgen.
