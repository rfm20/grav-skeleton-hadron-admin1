**Modules**
This modular page consists of 3 modules: a hero module and two text modules. Each module can be configured individually and contains its own content. This allows for complex page structures to be easily realized.

**Hero Module**
The hero module is a simple module that contains a background image, a title, and a subtitle. It can optionally be styled with a parallax effect or other styles.
